# Souzadex

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A **Souzadex** é uma galeria colaborativa, uma pokédex, de fotos do Souza
(Vitor Souza), geradas por IA.

Cada Souza da galeria foi enviado por alguém da comunidade via pull request.

**Galeria:** https://souzadex.vercel.app/

## Como gerar seu Souza

Acesse um modelo de prompt para gerar seu souza em [PROMPT.md](Prompt.md)

## Como contribuir

Para contribuir com o projeto, confira as instruções completas em [CONTRIBUTING.md](CONTRIBUTING.md).
